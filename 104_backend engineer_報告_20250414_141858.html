Backend Engineer 職缺分析報告

執行摘要

根據搜索關鍵詞 "backend engineer"，共找到 66 個相關職缺。市場需求旺盛，職缺分佈於台北市、新北市等地區，薪資範圍廣泛。

薪資分析

薪資範圍從待遇面議至月薪 85,000~150,000元不等，高薪職缺多集中在科技新創公司及跨國企業，低薪職缺則較偏向Junior級別。

技能需求

最常見的技能要求包括 CI/CD、Golang、RESTful API、Python 等，對後端開發、溝通能力及團隊合作有著明確要求。

地區分布

職缺多集中在台北市內湖區、松山區，新北市中和區，地區間薪資水平及技能需求有所差異。

推薦機會

  1. Golang 後端工程師（Server API / WebSocket） - 薩摩亞商創威有限公司，提供高效API和WebSocket開發機會。
  2. 資深後端工程師 Senior Backend Engineer - 春樹科技股份有限公司，參與知名發票對獎APP開發。
  3. 【Node.js/Typescript】資深後端工程師 Senior Backend Engineer - 鷹翔有限公司，提供高度客製化解決方案的機會。

求職建議

建議求職者多學習 CI/CD、Golang 等熱門技能，注重技術深度與團隊合作能力的提升，積極關注市場動向及公司特色。
